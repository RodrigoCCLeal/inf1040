Para Compilar todos os arquivos:

Baixe a biblioteca Make:
https://community.chocolatey.org/packages/make

Rode o comando no terminal:
"make"

Para Rodar todos os casos de teste ao mesmo tempo use o seguinte comando no terminal:
".\rodar_testes.bat"

Para rodar a aplicação como um inteiro use o seguinte comando no terminal:
".\foodies.exe"
